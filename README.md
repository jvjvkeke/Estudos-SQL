# 📊 Estudos SQL

Repositório com meus estudos de banco de dados utilizando MySQL.

## 📚 Sobre
Este projeto contém scripts desenvolvidos durante meu aprendizado em SQL, com foco em comandos básicos de manipulação e estrutura de dados.

## 📁 Estrutura

- /sql/schema.sql → criação das tabelas  
- /sql/alteracoes.sql → alterações na estrutura  
- /sql/dados.sql → inserção e manipulação de dados (CRUD)

## 🧠 Conteúdos praticados

- CREATE TABLE  
- ALTER TABLE  
- INSERT  
- SELECT  
- UPDATE  
- DELETE  
- TRUNCATE  

## 📚 Referência
Estudos baseados nas aulas do Curso em Vídeo, com o professor Gustavo Guanabara.
